# SQL Navigator

Упрощает навигацию по логам, комментариям, и другим меткам в коде.
 

## 🔧 Возможности

- Автоматическое сканирование SQL-файлов на наличие комментариев
- Отображение списка комментариев в боковой панели VS Code
- Переход к строке с комментарием одним кликом
- Поддержка кастомных шаблонов комментариев (например: `--*`, `lgLog`, `--!`, `--` и любых других).
- Быстрая вставка шаблонного комментария: если задано несколько шаблонов, будет предложено выбрать нужный.
 

## ⚙️ Настройки

- `sqlNavigator.enabled` – включить/отключить расширение.
- `sqlNavigator.commentPatterns` – **новое поле** (массив строк). Список шаблонов меток для поиска. По умолчанию `["--*"]`.  
  Примеры: `["--*", "lgLog", "--?"]` – будут найдены строки, начинающиеся с любого из этих шаблонов.
